# Monitoramento & Performance

**`Analista de Monitoramento | BI | Automações`**

Atuo com monitoramento operacional, análise de dados e automação de processos. 
Meu foco é transformar dados operacionais em informação acionável, garantindo controle, eficiência e rastreabilidade.

Tenho experiência prática com integração de sistemas, construção de indicadores e desenvolvimento de soluções para suporte à operação.

---

### Principais Atividades

- Monitoramento operacional com análise de eventos e desvios
- Construção de indicadores e dashboards (Power BI)
- Integração de APIs (Microsoft Graph, Fluig, etc.)
- Automação de processos com Python
- Estruturação de regras de negócio e penalidades operacionais

---

### Projetos em Destaque

- **Sistema de Penalidades**  
  Gestão e aplicação automatizada de penalidades operacionais com rastreabilidade

- **Integração + Power BI**  
  Extração, tratamento e disponibilização de dados para análise

- **Monitoramento com IA (Wisevia)**  
  Identificação de comportamentos de risco via vídeo e telemetria

- **Extratores e Automações**  
  Scripts para coleta e processamento de dados operacionais
